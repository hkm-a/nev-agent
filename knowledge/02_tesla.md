# Tesla (特斯拉)

特斯拉是全球领先的纯电动汽车制造商，总部位于美国德州奥斯汀。核心优势：FSD(Full Self-Driving)智驾系统、4680电池、一体化压铸技术、超充网络(全球超50,000个超充桩)。

## 主力车型

### Model 3 (焕新版)
- 定位：中型纯电轿车
- 价格：24.59-33.59万元
- 续航(CLTC)：606-713km
- 零百加速：3.1s(Performance版)
- 电池：磷酸铁锂(后驱) / 三元锂(长续航/Performance)
- 智驾：HW4.0硬件，基础AP标配，EAP/FSD选装(FSD中国区6.4万元)
- 亮点：2024焕新版，后排屏幕，通风座椅，双层隔音玻璃

### Model Y
- 定位：中型纯电SUV
- 价格：26.39-36.89万元
- 续航(CLTC)：554-688km
- 零百加速：3.7s(Performance版)
- 电池：磷酸铁锂(后驱) / 三元锂(长续航/Performance)
- 特点：全球销冠车型(2023年)，超大后备箱(2158L放倒)
- 智驾：HW4.0，同Model 3

### Cybertruck
- 定位：纯电皮卡(中国暂未上市)
- 价格：US$60,990起
- 续航：515-708km(里程扩展包)
- 特点：超硬不锈钢车身，装甲玻璃，支持48V电气系统
- 零百：2.6s(Cyberbeast版)
