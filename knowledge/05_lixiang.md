# 理想 (Li Auto)

理想汽车是中国增程式电动汽车领军品牌，总部位于北京。核心特色：增程电动(EREV)技术、"移动的家"产品理念、家庭用户定位、理想AD智驾系统。

## 主力车型

### L6
- 定位：中大型增程SUV(五座)
- 价格：24.98-27.98万元
- 续航：纯电CLTC 212km / 综合1390km
- 零百加速：5.4s
- 电池：磷酸铁锂(36.8kWh) + 1.5T增程器
- 智驾：AD Pro(地平线J5) / AD Max(双Orin-X)

### L7
- 定位：中大型增程SUV(五座)
- 价格：30.18-37.98万元
- 续航：纯电CLTC 225km(WLTC 175km) / 综合1315km
- 零百：5.3s
- 电池：三元锂(42.8kWh/52.3kWh) + 1.5T增程器
- 特点：后排大空间，皇后座，15.7英寸后排娱乐屏

### L8
- 定位：中大型增程SUV(六座)
- 价格：32.18-39.98万元
- 续航：纯电CLTC 225km / 综合1315km
- 零百：5.3s
- 特点：理想ONE换代，六座独立座椅，冰箱+大电视

### L9
- 定位：大型增程SUV(六座)
- 价格：42.98-43.98万元
- 续航：纯电CLTC 235km(WLTC 180km) / 综合1315km
- 零百：5.3s
- 电池：三元锂(52.3kWh) + 1.5T增程器
- 亮点：后排冰箱(0-7°C)，后排大屏(电动收起)，空气悬架+CDC，OLED中控屏
- 智驾：AD Max(双Orin-X+激光雷达)
